## Troubleshooting

**The extension does not detect any media on the page**
Make sure the video is playing before you open the popup. The extension needs to see network activity from the stream to detect it.

**The download starts but fails partway through**
Check your internet connection and try again. A stable connection is required for the entire download process.

**The popup does not open when I click the icon**
Try refreshing the SaintPorn page and clicking the icon again. If the issue persists, reinstall the extension from the latest release.

**I see an error about unsupported video format**
Only videos played directly on SaintPorn video pages are supported. Embedded or external players may not work.

**The download is very slow**
Large video files can take time to download. Make sure you have a fast internet connection and close other bandwidth-heavy applications.
